## Overview

Welcome to your Palworld server! This guide will walk you through the initial setup and configuration of your Palworld dedicated server on Infected Games Hosting.

## Prerequisites

Before you begin, ensure you have:

- An active Palworld server from Infected Games Hosting
- Access to your [Game Panel](https://panel.infectedgamehosting.com)
- Basic understanding of server management

## Initial Server Setup

### Step 1: Access Your Server

1. Log in to your [Game Panel](https://panel.infectedgamehosting.com)
2. Select your Palworld server from the server list
3. Wait for the server to finish installing (first-time setup)

### Step 2: Start Your Server

1. Navigate to the **Console** tab
2. Click the **Start** button
3. Monitor the console for startup messages
4. Wait for the message: `Server is ready`

Your server typically takes 2-5 minutes to start up completely.

### Step 3: Connect to Your Server

Once your server is running:

1. Launch Palworld on your gaming PC
2. Select **Join Multiplayer Game**
3. Click **Connect to Server**
4. Enter your server IP and port (found in your game panel)
5. Click **Connect**

## Server Information

You can find your server connection details in the game panel:

- **Server IP**: Displayed at the top of your panel
- **Server Port**: Default is `8211` (Game Port)
- **RCON Port**: Default is `25575` (Admin Port)

## Common Issues

### Server won't start

If your server fails to start:

1. Check the console for error messages
2. Verify the server has finished installing
3. Try stopping and restarting the server
4. Contact support if the issue persists

### Can't connect to server

If you can't connect:

1. Verify the server is running (green status indicator)
2. Check you're using the correct IP and port
3. Ensure your firewall isn't blocking the connection
4. Verify the server isn't password protected (unless intended)
