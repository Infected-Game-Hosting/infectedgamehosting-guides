TROUBLESHOOTING GUIDE - INFECTED GAME HOSTING

Common Palworld Server Issues and Solutions:

SERVER WON'T START OR KEEPS CRASHING:

Possible Causes:
- Mod compatibility issues
- Corrupted server files
- Insufficient memory allocation
- Configuration file errors
- World file corruption

Troubleshooting Steps:
1. Check server status in your control panel at https://billing.infectedgamehosting.com/
2. Review server logs for error messages
3. Verify mod compatibility through your dashboard
4. Try restarting server through control panel
5. If issue persists, contact support with error logs

IGH Resolution:
- Our team can quickly identify the root cause
- Automated mod conflict detection available
- Emergency server restoration from backups
- Professional diagnosis within 2-4 hours
- Permanent fix implementation

PLAYERS CAN'T CONNECT TO SERVER:

Connection Issues Checklist:
1. Verify server is running (green status in control panel)
2. Check that you're using the correct IP address and port
3. Confirm server isn't full (check player count vs limit)
4. Verify firewall settings on player devices
5. Test connection from different networks

Server-Side Checks:
- Server status: Must show "Online" or "Running"
- Port configuration: Ensure correct ports are open
- Player slots: Verify available player slots
- Mod requirements: Check if players need specific mods
- Whitelist settings: Confirm player is on whitelist (if enabled)

Player-Side Solutions:
- Verify Palworld client is updated to latest version
- Check Windows/Mac firewall settings
- Try direct IP connection instead of server browser
- Restart Palworld client completely
- Verify Steam is running and connected

IGH Network Diagnostics:
- Real-time connectivity testing available
- Network path analysis tools
- Automatic port configuration
- DDoS protection status verification
- Latency and packet loss monitoring
